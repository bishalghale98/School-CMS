School Website CMS

A School Website CMS is a web-based content management system that enables schools to manage and update their official website without technical knowledge. It provides a modern public website for visitors and a secure Filament Admin Panel where administrators can manage content, notices, admissions, events, galleries, and other website information.

Project Goal

The goal of this project is to provide schools with a modern, responsive, and easy-to-manage website where administrators can update content through a user-friendly CMS. The system focuses on managing the school's online presence, publishing announcements, showcasing activities, handling simple online admission inquiries, and maintaining communication with students, parents, and visitors, without including complex School Management System (SMS) features such as attendance, examinations, fee management, or student records. This makes the application lightweight, maintainable, and suitable for schools of all sizes.
